# Linux commands

## Basic commands

```bash
ls  # list  
cd Lab/ # mi sposto nella cartella Lab  
cd ..  # sposto una cartella indietro  
mkdir Lab1 # creo la cartella Lab1  
pwd  # display del path  
cat Lab1/lab1.txt  # visualizzo il cotenuto del file lab1.txt  
mv Lab Lab2 # rinomino la cartella Lab in Lab2  
touch lab2.txt # creo il file lab2.txt  
nano lab3.txt  # creo il file lab3.txt  
echo "CIAO"  # display stringa "CIAO"  
echo "Ciao" >> [Lab1/lab1.txt] # redirigo e salvo la stringa nel file lab1.txt  
echo "Pippo" > lab2.txt # salvo la stringa "Pippo" nel file lab2.txt cancellando l'eventuale contenuto  
ls Lab1/ && cat Lab1/lab1.txt # list della cartella Lab1 e visualizza il contenuto del file lab1.txt  
head Lab1/lab1.txt # visualizza le prime righe del file lab1.txt    
head -3 Lab1/lab1.txt # visualizza le prime n righe (in questo caso 3) del file lab1.txt  
```
